# 文档注释

## 目标

为 Carven 添加文档注释语法 `//!` 和 `///`，用于从源码生成文档。

## 提议语法

```carven
//! 模块级文档注释（描述当前模块的功能和用途）

/// 函数级文档注释（描述下方声明的函数）
fn add(a: i32, b: i32) -> i32 {
    return a + b;
}

/// 结构体文档注释
struct Point {
    /// 字段文档注释
    x: f64,
    y: f64,
}
```

## 规则

- `//!` — 模块/文件级文档，描述当前模块
- `///` — 项目级文档，描述紧随其后的声明（函数、结构体、枚举等）
- 内部支持 Markdown 格式

## 转译策略

- 作为特殊注释嵌入生成的 C++ 代码中
- 未来可用于自动生成文档（类似 `rustdoc`）
