# Roadmap

Implemented features are documented in [grammar.md](grammar.md).

## In Progress

| Feature | Status |
|---------|--------|
| Range-for loops `for i in 1..10` | proposed |
| Doc comments `///` / `//!` | proposed |
| Inline tests | proposed |
| Inline C++ blocks | proposed |

## Planned

- Struct literals (`Point { x: 1, y: 2 }`)
- Generics / templates
- Traits / interfaces
- Lambdas
- Enum variants with payloads
- Module-level `const` and `static`
- `extern` function declarations
- Multi-file projects
- `export` for module authors
- `as` type casts
