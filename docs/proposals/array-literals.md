# 数组字面量

## 目标

支持在 Carven 中直接书写数组字面量，转译为 `std::array`。

## 语法

### Carven

```carven
let a = [1, 2, 3];
let b: [i32; 3] = [1, 2, 3];  // 可选：带类型标注
```

### 转译输出

```cpp
const auto a = std::array { 1, 2, 3 };
const auto b = std::array<std::int32_t, 3> { 1, 2, 3 };
```
